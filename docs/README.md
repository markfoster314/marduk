# Doc Assets

This directory contains assets used in documentation, including README.md and other markdown files.

## Image Guidelines

### Naming Conventions

Use lowercase with hyphens for all image files:

- ✅ `button-variants.png`
- ✅ `component-demo.gif`
- ✅ `dark-theme-example.jpg`
- ❌ `ButtonVariants.png`
- ❌ `component_demo.gif`

### Optimization

Always optimize images before committing:

```bash
# For PNG files
# Use tools like TinyPNG, ImageOptim, or pngquant

# For GIF files
# Use tools like Gifski, ezgif.com, or gifsicle

# For SVG files
# Use SVGO or SVGOMG
```

---

**Tip**: Consider using [Shields.io](https://shields.io/) for badges instead of static images when possible.
